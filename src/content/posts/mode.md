## 1. ECB 模式

## 2. CBC 模式
> Cipher Block Chaining mode (密码分组链接模式) 

## 3. CFB 模式
> Cipher FeedBack mode (密文反馈模式) 

## 4. OFB 模式
> Output FeedBack mode (输出反馈模式) 

## 5. CTR 模式
> CounTeR mode (计数器模式)
